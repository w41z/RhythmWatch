### Clone the repository
---
`git clone git@github.com:w41z/RhythmWatch.git`

### Warning
---
Please start the work with new branches, **DO NOT MODIFY THE `main` BRANCH DIRECTLY!!!**
You can use the following command to create a new branch and switch to it:
```
git branch <name_of_branch>
git switch <name_of_branch>
```
